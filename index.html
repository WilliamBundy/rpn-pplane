rpn-pplane

Written for an ODE class, this is a simple replacement for the java versions found here: http://math.rice.edu/~dfield/dfpp.html

Equation format

Equations are given in reverse polish notation, meaning that the operator comes after the operands.

For example: x + 1 becomes x 1 +

Constants

Terms that don't match any defined functions or variables are treated as constants and parsed by haXe's Std.parseFloat function. All positive and negative real numbers work, anything else gets treated as not-a-number (NaN).

Variables

The only defined variables are x and y. Constructs such as 2x are not allowed, use 2 x *.

Defined Functions:

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /

  • Powers: ^

  • Square Root: sqrt

  • Sine: sin
  • Cosine: cos
  • Tangent: tan
  • Natural Logarithm: log
